# vue_mall

## 介绍
 
【vue移动端购物街项目】 目前实现了主页跳转详情页,上拉加载更多商品,以及切换商品列表,详情页导航栏电梯导航,添加商品至购物车，购物车结算已选商品总价等基础功能


#### 注意

该页面属于SPA页面,基于vue,vuex,vue-router,better-scroll,fastclick等完成,数据来自后台的接口,目前尚未实现登录注册等实际功能

## 运行步骤

#### 1、安装必要的包

npm install

#### 2、本地运行

npm run serve

#### 3.项目打包部署

npm run build
